# Dev Senior - CODIFICO

## Game - Batalla Naval

### Requerimiento
Construir una aplicación para dos personas basado en el famoso juego de **Batalla Naval** utilizando tecnologías Microsoft en back-end y cualquier tecnología en front-end.

### Aclaraciones
Para el correcto desarrollo de la prueba, es necesario que tenga en cuenta los siguientes puntos:
+ **Clone** este repositorio.
+ Cree un proyecto privado de **GitLab** (free) o un repositorio privado de **GitHub** ($).
+ Realice un push al nuevo remoto y efectúe sus commits ahí. 
+ Otorgue permisos y acceso a **@CJavierSaldana** para tener acceso al código y realizar las pertinentes observaciones.
+ Desarrolle su proyecto en la solución `Codifico.Senior.sln`
+ Sí cree necesario agregar más proyectos, por favor agréguelos bajo el folder `source/`
+ Tenga en cuenta que la solución será compilada en otras máquinas. Asegúrese de no tener referencias locales.

### Bonus
+ Usted puede utilizar todas las tecnologías que vea conveniente para cumplir con el requerimiento. No obstante, la utilización de algunas en particular darán puntos adicionales a la hora de la revisión del código:
   + SignalR.
   + Angular.
   + Bootstrap.
+ Código limpio.
+ Aplicar arquitectura de la aplicación (front y back).
+ Seguir principio SOLID.
+ Codificación en inglés. 
+ Realice distintos commits evidenciando el progreso.

### Mecánicas
+ Los jugadores `1` y `2` tendrán distintas sesiones en el navegador.
+ Los jugadores no podrán ver las posiciones de los barcos de su oponente.
+ El tablero debe ser de `8 x 8`.
+ Barcos disponibles
   + Dos barcos de `1 x 2` 
   + Un barcos de `1 x 3` 
   + Un barcos de `1 x 4` 
+ La asignación de los barcos en el tablero debe ser aleatoria.
+ La jugabilidad debe ser por turnos.
+ Declarar un ganador y dar la opción de reiniciar el juego.
